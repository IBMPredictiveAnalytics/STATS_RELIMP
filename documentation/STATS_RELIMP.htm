Relative Importance Measures for Regression

This command calculates various relative importance measures for regression predictor variables

Dependent Variable Select a scale-level dependent variable.

Forced Variables Optionally select variables that will always be included in the regression. Importance measures are not calculated for these variables.

Independent Variables Select one or more predictor variables whose importance measures should be calculated.

Any categorical independent or forced variables are automatically converted to factors, and the importance measures for non-forced variables are reported for the entire factor.

Weight The standard Statistics weight setting is not used. Weights can be replication weights, heteroscedasticity weights, or inverse probability of selection weights.

Importance Metrics Select one or more importance measure to be calculated.

Shapley Value the incremental R2 for the variable averaged over all models. This is referred to as LMG in the output.

First The R2 for the variable when entered first

Last The incremental R2 when the variable is entered last

Beta Sq The square of the standardized coefficient. These values would sum to the R2 if the independent variables were uncorrelated.

Pratt The standardized coefficient times the correlation

CAR Marginal correlations after adjusting for correlations among the regressors.

Options

Scale relative importance measures to 100% All of the importance measures decompose the equation R2 into parts except for First and Last. Check this box to scale the importance measures to sum to 100%

Display regressor ranks Check this box to display a table showing the importance rank of each of the non-forced regressors for each importance measure

Plot relative importance Check this box to display a bar chart of the importance measures for each measure. Variable names for some bars will be omitted if there is insufficient space for them, but the bars are in the same order as the non-forced regressors. The names may be truncated.

Missing Values Select listwise to delete cases with missing values or Stop to terminate the procedure if missing values are found.

This procedure is computationally intensive and can take a long time if there are many independent variables.

Note: Occasionally the computations may fail with an assertion of singularity when the regressor matrix is not actually singular. It may help in such a case to rescale variables with very large values.

Additional Features

This dialog generates syntax for the STATS RELIMP extension command. To display help for this command, run the following syntax:

STATS RELIMP /HELP.

One additional importance metric is only available in the non-US version of the relaimpo package which must be obtained from http://prof.beuth-hochschule.de/groemping/relaimpo and is licensed for use only outside the United States. For this reason, this extension command has not been tested with this option. This option is not supported in the dialog box but can be specified in the MEASURE list.

PMVD - the proportional marginal variance decomposition as proposed by Feldman It can be interpreted as a weighted average over orderings among regressors, with data-dependent weights

Requirements

This command requires the R Essentials and the STATS RELIMP extension command. You can download these from the SPSS Community at www.ibm.com/developerworks/spssdevcentral. It also requires the R relaimpo package and at least Statistics Version 18. The relaimpo package will be automatically downloads during the command installation if possible.
